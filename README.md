# Vision Garden · Phase 5 — Developer Landing Page

A cinematic, single-page promotional website for **Vision Garden 築夢園 · Phase 5**, a limited
collection of 43 double storey terrace houses in Taman Wawasan Mahandoi, Donggongon, Penampang,
Sabah — developed by **Sri Moraine Sdn Bhd**.

Built from the official sales brochure and cinematic promo film.

## Highlights

- **Cinematic hero** with the exterior artist's impression and a "Watch the film" call-to-action
- **Embedded promo film** (`assets/video/vision-garden-promo.mp4`) with a custom poster & play overlay
- **Animated stat strip** — 43 units · 4 bedrooms · 20ft frontage · 99-year leasehold
- **Interactive floor plans** — tabbed viewer (Intermediate & Corner, Type F / Type G) with click-to-enlarge lightbox
- **Full building specifications**, unit comparison, and location / masterplan section
- **Register-interest form** with client-side validation and a success state
- Fully **responsive** (mobile hamburger nav), scroll-reveal animations, and complete legal disclaimers

## Tech

Pure, dependency-free **HTML + CSS + JavaScript** in a single `index.html`.
Only external resource is Google Fonts (Cormorant Garamond + Jost). All imagery is optimised WebP.

## Run locally

```bash
python -m http.server 8000
# then open http://127.0.0.1:8000
```

## Deploy (GitHub Pages)

Settings → Pages → Deploy from branch → `main` / root. The site is served as-is.
